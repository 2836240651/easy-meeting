🔧 技术实现

1. 后端查询逻辑

SQL查询条件确保返回正确的消息：

WHERE (m.receive_user_id = #{userId} AND m.receive_type = 1)
  OR m.receive_type = 0

这个条件会返回：

  • 所有群聊消息（receiveType=0）
  • 发送给当前用户的私聊消息（receiveType=1）

2. 前端消息处理

// 判断是否为私聊消息
const isPrivate = msg.receiveType === 1

// 获取接收者信息
let receiveUserName = ''
if (isPrivate && msg.receiveUserId) {
  const receiver = participants.find(p => p.userId === msg.receiveUserId)
  receiveUserName = receiver ? receiver.name : '未知用户'
}

3. CSS样式区分

/* 私聊消息样式 */
.chat-message.private-message {
  background-color: rgba(52, 152, 219, 0.1);
  border-left: 3px solid #3498db;
}

/* 自己发送的消息 */
.chat-message.my-message .message-content {
  background-color: #2c5282;
  border-color: #3182ce;
}

📝 使用说明

  1. 查看聊天历史：打开会议后点击"聊天"按钮，系统会自动加载历史消息
  2. 识别消息类型：
    • 普通消息 = 群聊消息（所有人可见）
    • 带"私聊"标签的消息 = 私聊消息（只有发送者和接收者可见）
  3. 查看发送者：每条消息左侧显示发送者头像和名称
  4. 私聊接收者：私聊消息会显示"→ 接收者名称"
  5. 自己的消息：自己发送的消息会有蓝色背景高亮
