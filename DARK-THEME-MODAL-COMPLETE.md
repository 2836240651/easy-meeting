## 技术实现

### 使用 :deep() 选择器
在组件的 scoped 样式中使用 `:deep()` 选择器来修改 Element Plus 组件的内部样式：

```css
:deep(.el-dialog) {
  background-color: #1a1a1a;
}
```

### 全局样式覆盖
对于弹出层（如日期选择器、下拉菜单），使用全局样式文件覆盖：

```css
.el-select-dropdown {
  background-color: #2a2a2a !important;
}
```

### 重要性声明
使用 `!important` 确保样式优先级高于 Element Plus 的默认样式。
