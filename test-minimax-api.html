MiniMax API 可能的问题: 1. API Key 格式不正确 2. API URL 不正确 3. 模型名称不正确 4. 需要额外的请求头 5. CORS 限制(浏览器直接调用会失败) 常见的 MiniMax API 格式: - URL: https://api.minimaxi.com/v1/chat/completions - 或: https://api.minimax.chat/v1/text/chatcompletion - Header: Authorization: Bearer YOUR_API_KEY - 或: api-key: YOUR_API_KEY 请查看 MiniMax 官方文档确认正确的 API 格式。
