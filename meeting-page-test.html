  • ⚠点击"会议详情"按钮，查看会议信息浮框
  • ⚠点击"设置"按钮，切换设置选项卡
  • ⚠点击底部控制按钮，测试静音、视频等功能
  • ⚠点击"成员"按钮，查看参与者列表
  • ⚠点击"聊天"按钮，发送测试消息
  • ⚠测试键盘快捷键功能
  • ⚠测试全屏模式切换

  • 框架：Vue 3 Composition API
  • 路由：Vue Router
  • API：Axios HTTP客户端
  • 样式：Scoped CSS，响应式设计
  • 图标：SVG图标集成
  • 主题：深色主题 (#434343, #363636, #999999)
  • 兼容性：现代浏览器，移动端适配
